# Federated Wiki - ThoughtStorms *wikish* Plugin

Plugin type for "Wikish", the markup language of the ThoughtStorms wiki and SdiDesk. It's derived from [UseMod](http://www.usemod.com/cgi-bin/wiki.pl) format.

## Build

    npm install
    grunt build
